初学者向けポイント解説

argparse：コマンドのオプションを受け取る標準機能

secrets.choice()：リスト（文字集合）から安全に1文字取る

各カテゴリ最低1文字：実運用でよくある要件（強度の底上げ）

シャッフル：先頭が固定パターンにならないように最後に混ぜる

まとめ：ここで学べること
概念	説明
secrets.choice()	セキュアな乱数で文字を選ぶ
argparse	実行時オプションを受け取る方法
string	英字・数字・記号の定義を簡単に扱う
join()	リストを文字列にまとめる
for ... in range()	リストの要素を順に処理
raise ValueError	不正な入力を防ぐ（安全設計）
